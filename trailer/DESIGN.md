# Design System

## Overview

《沫沫的珊瑚午餐》以横版海底 Canvas 为主体，使用深海蓝的大面积环境、暖橙角色和贝壳装饰 UI 建立可爱像素冒险气质。界面结构由顶部轻量 HUD、中央装饰对话框和全屏水下场景构成。视觉重点始终落在戴头灯的橙白小丑鱼“沫沫”，周围以低对比蓝色岩层、气泡和焦散保持空间感。

## Colors

- **Deep Ocean**: `#062F80` — 主海水背景。
- **Abyss Surface**: `#021239` — 深色遮罩、片尾与文字承托面。
- **Panel Navy**: `#031F45` — 对话框和深色 UI 面板。
- **Foam Cyan**: `#B3F8ED` — 正文、HUD 与水下高光。
- **Ice Highlight**: `#E2FDFF` — 最亮反光与描边。
- **Sun Gold**: `#FFD668` — 星星、成长状态和主要按钮。
- **Shell Orange**: `#FFAD55` — 暖色按钮与贝壳装饰。
- **Coral Orange**: `#F47C4C` — 标题重点、危险提示和能量峰值。
- **Coral Outline**: `#793741` — 暖色按钮与珊瑚图形的深描边。
- **Water Accent**: `#5CC0E6` — 水流、玻璃 HUD 与光束高光。

## Typography

- **Primary**: PingFang SC；无本地字体文件时使用系统中文无衬线字体栈。
- **Display**: 700；中文主标题，约 58px 网页基准，宣传片中按 1080p 放大到 84–128px。
- **Body**: 650；中文旁白字幕和说明文字，保持清晰圆润。
- **Label**: 650–700；HUD、关卡和短标签，轻微字距。
- **English Kicker**: 650；全大写、小尺寸、高字距，只作为辅助信息。

## Elevation

空间主要来自三层蓝色景深、聚光灯和焦散，不使用写实投影。HUD 采用半透明蓝色玻璃层、轻度 backdrop blur、青白内描边与深蓝下沿；暖色按钮使用橙黄渐变、酒红描边和短距离像素阴影。角色与 UI 图标保留深海军蓝像素轮廓，避免在蓝色背景里消失。

## Components

- **Shell Dialogue Frame**: 珊瑚、贝壳和海星构成的像素边框，内部是深蓝面板。
- **Glass Growth HUD**: 顶部横向成长条，深蓝轨道配金色百分比。
- **Bubble Hearts**: 青色泡泡心生命值，圆润、明亮、带水滴高光。
- **Pearl Round Controls**: 圆形玻璃按钮配贝壳/珍珠图标。
- **Sunset Action Button**: 黄到珊瑚橙的胶囊按钮，深色描边和短下投影。
- **Headlamp Hero**: 沫沫的铜色探照灯和锥形光束，是场景中的主要方向性光源。
- **Parallax Reef**: 远景海床、中景石拱/尖塔和前景岩架的蓝色像素层。

## Do's and Don'ts

### Do's

- 使用深蓝大面积环境，让金色、橙色和泡沫青承担视觉焦点。
- 保持最近邻像素边缘、深色描边和三分之四侧视角。
- 用气泡、焦散、漂浮颗粒和轻微视差维持水下生命感。
- 让短中文标题占据清晰层级，并为字幕保留低干扰区域。
- 对实录画面使用柔和推近、局部遮罩或图形强调，而不是硬裁切。

### Don'ts

- 不引入紫红霓虹、纯黑科技风或写实海洋摄影。
- 不使用细字、远程字体或明显比中文更粗的英文/数字。
- 不让高亮焦散压过角色和 HUD。
- 不拉伸像素素材，不使用模糊插值。
- 不用标准矩形白底字幕条或写实长阴影破坏游戏风格。
