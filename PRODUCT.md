# Product

## Register

product

## Users

以儿童、亲子用户和喜欢轻量网页小游戏的玩家为主。用户可能在手机浏览器中用单指短时游玩，也可能在桌面端使用鼠标或键盘持续探索。

## Product Purpose

提供一款打开即玩的海底成长冒险游戏。成功体验应让玩家不用阅读复杂说明就理解移动、冲刺、吃小鱼、躲大鱼、成长与挑战鲨鱼，并在三片有明显差异的海域中持续获得可爱、清晰的反馈。

## Brand Personality

可爱、沉浸、活泼。画面有传统像素游戏的手工感，同时保留柔和的海水光影和儿童友好的情绪。

## Anti-references

避免通用 AI 仪表盘、厚重的纯色线框卡片、硬质科幻切角、整张 UI 贴图拉伸、文字与装饰错位、过度玻璃模糊、桌面操作文案出现在触屏设备，以及为了塞下所有信息而把移动端 HUD 挤成难读的多行布局。

## Design Principles

1. 游戏画面优先，HUD 只保留当前决策需要的信息。
2. 触屏体验不是桌面版缩小，而是重新分组信息并使用触屏语言。
3. 视觉体型、碰撞、光源和反馈必须彼此一致。
4. CSS 负责弹性布局，透明像素素材负责角色与语义识别，装饰不能破坏适配。
5. 所有修改必须在真实浏览器、多方向和多尺寸中复验。

## Accessibility & Inclusion

交互目标至少 44×44 CSS px；纯图标按钮保留可读名称；核心状态不只依赖颜色；支持减少动态效果；中英文和数字保持清晰字重；触屏、鼠标和键盘均可完成核心玩法。
